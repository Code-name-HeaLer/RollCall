## 🚀 Quick Start

### Prerequisites
- Node.js 18+ and npm/yarn
- Expo CLI (`npm install -g @expo/cli`)
- iOS Simulator (Mac) or Android Studio (for emulators)
- Expo Go app on your phone (for testing)

### Installation

```bash
# Clone the repository
git clone https://github.com/Code-name-HeaLer/RollCall.git
cd RollCall

# Install dependencies
npm install

# Start the development server
npx expo start
```

### Running the App

Choose your preferred method:

**📱 On Your Device:**
```bash
# Scan QR code with Expo Go app
npx expo start
```

**🤖 Android Emulator:**
```bash
# Press 'a' in terminal or
npx expo start --android
```

**📱 iOS Simulator:**
```bash
# Press 'i' in terminal or
npx expo start --ios
```

### First Time Setup

1. **Launch the app** on your device or emulator
2. **Add your first subject** from the Subjects tab
3. **Set up your timetable** in the Schedule section
4. **Configure notifications** in Settings
5. **Start tracking** your attendance!

## 📖 Documentation

### 🏗️ Project Structure
```
RollCall/
├── app/                    # Expo Router pages
│   ├── (tabs)/            # Tab navigation screens
│   ├── modals/            # Modal screens
│   └── _layout.tsx        # Root layout
├── components/            # Reusable UI components
├── services/             # Database and business logic
├── types/                # TypeScript type definitions
├── constants/            # App constants and themes
└── assets/               # Images, fonts, and static files
```

### 🔧 Configuration

#### Environment Setup
Create a `.env.local` file:
```env
EXPO_PUBLIC_APP_VERSION=1.0.0
EXPO_PUBLIC_API_URL=your-api-endpoint
```

#### Customizing Themes
Edit `constants/Colors.ts`:
```typescript
export const Colors = {
  light: {
    primary: '#6366f1',
    background: '#ffffff',
    // ... your custom colors
  },
  dark: {
    primary: '#818cf8',
    background: '#0f0f23',
    // ... your custom colors
  }
}
```

### 🧪 Testing

```bash
# Run tests
npm test

# Run with coverage
npm run test:coverage

# E2E tests
npm run test:e2e
```

**Q: App crashes on startup**

A: Check your device's notification settings and ensure RollCall Pro has permission to send notifications.
